## Codebase Patterns
- pyannote segmentation model input: (batch, 1, num_samples) float32 at 16kHz; output: (batch, num_frames, num_speakers) float32
- WeSpeaker embedding model input: (batch, num_frames, 80) fbank features; output: (batch, embedding_dim) float32
- PLDA files: xvec_transform.npz has 'mean' and 'transform'; plda.npz has 'Phi' and 'Sigma' (note capitalization in Python source)
- Pipeline clustering attr: pipeline.klustering (not pipeline.clustering)
- Pipeline segmentation model attr: pipeline._segmentation.model_
- WeSpeaker ONNX model available at hbredin/wespeaker-voxceleb-resnet34-LM as speaker-embedding.onnx
- ort v2: requires `std` + `tls-native` features with `download-binaries`; ort::Error<SessionBuilder> not Send+Sync — use .map_err() not `?` with anyhow
- .npz parsing: use zip crate to extract entries, npyz::NpyFile to parse individual .npy arrays into ndarray
- Diarization models cached at ~/.cache/fast-whisper-rs/diarization/ (XDG_CACHE_HOME respected)
- ort v2: Session::run takes `&mut self`; use TensorRef::from_array_view(&owned_array) for inputs; try_extract_tensor returns (&Shape, &[f32]) where Shape derefs to [i64]
- ort v2 ndarray version: ort 2.0.0-rc.12 uses ndarray 0.17 — must match in Cargo.toml
- pyannote aggregation: skip_aggregation=True in diarization pipeline, binarization happens on per-chunk outputs BEFORE aggregation
- Fbank features: frame=400 samples (25ms), hop=160 (10ms), pre-emphasis=0.97, Hamming window, radix-2 FFT, 80 mel triangular filters, log energy
- WeSpeaker ResNet34 default embedding dim: 256
- AHC clustering: average linkage with cosine distance on PLDA-transformed embeddings, threshold 0.6
- PLDA transform: center (subtract mean), apply LDA matrix, truncate to lda_dim (128), L2 normalize
- Export script PLDA files: xvec_transform.npz stores 'mean' + 'transform' from pipeline.klustering.preprocessing_transform_params_; plda.npz stores 'phi'+'sigma' from plda_params_ (lowercase keys despite Python using 'Phi'/'Sigma')
